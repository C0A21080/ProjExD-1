# 第4回
## 逃げるこうかとん(ex04/dodge_bomb.py)
### ゲーム概要
- ex04/dodge_bomb.pyを実行すると、1600x900のスクリーンに草原が描写され、こうかとんが爆弾と接触するとゲームが終わる。
### 操作方法
- 矢印キーでこうかとんを上下左右に移動する
### 追加機能
- 当たり判定のない偽爆弾が表示される
- 動かない偽のこうかとんを表示する
- 星が表示されるようになり、こうかとんに接触するとこうかとんが無敵となり、爆弾と接触してもゲームが終了しなくなる。
- こうかとんの初期座標がランダムになる
